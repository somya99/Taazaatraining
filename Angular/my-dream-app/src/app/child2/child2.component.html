This is a demo of Component Communication among components that are not related to each other. Here declared two components one is having a input box to accept the price entry and a button to set the price. ngModel is used for data binding that would be helpful in setting the price of entry which is declared in another component. Services is used to communicate among two components. When price get set in input parameter of a component that will get updated in the prices of entry displayed in another component.
