Hello welcome to Attribute Directive


This is a demo of Attribute Directive in Angular. The main aim is to set color of text along with border when mouse is hovered over it. It is achieved with the help of service. Host is div, Host binding is used for border and Host Listener is used for click, mouse enter and mouse leave events. Color of text is set by default to green and user can change it using input parameter in the display.
